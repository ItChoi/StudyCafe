# 📚 스터디카페 애플리케이션 - Backend

# 📌 서비스 소개

스터디카페 애플리케이션은 스터디카페 이용자의 좌석 예약 및 이용 관리, 운영자의 매장 관리 기능을 통합 제공하는 백엔드 기반 서비스입니다.  
이용자는 지점 및 좌석 정보를 실시간으로 확인하고 예약과 이용 현황을 관리할 수 있으며, 운영자는 좌석 운영 상태와 이용 데이터를 효율적으로 관리할 수 있습니다. 서비스는 안정적인 예약 처리와 확장 가능한 구조를 목표로 설계되었으며, 실제 운영 환경을 고려한 인증, 데이터 관리, API 구조를 중심으로 구현되었습니다.

---

# ✨ 주요 기능
## 이용자
- 지점 및 좌석 조회
- 좌석 예약 및 예약 취소
- 이용 현황 조회
- 시간 연장
- 좌석 이동
- 이용권 구매
  - 자유석
  - 고정석
  - 스터디룸
- 알림

## 관리자
- 좌석 배치
- 지점 및 좌석 관리
- 예약 및 이용 현황 관리
